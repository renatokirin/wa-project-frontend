

# Running the app

## Change backend url
/vue-app/src/config.js
`baseUrl: 'http://localhost:3000'` 


## Project Setup

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Compile and Minify for Production

```sh
npm run build
```
